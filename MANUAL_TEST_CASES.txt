R6-FIX 3 — Own review shows "Your review" label, not Report button  (was: R4-F1-A/C)
────────────────────────────────────────────────────────────────────────────────
Root cause: getCompanyReviews() always used the unauthenticated request()
helper → backend optionalAuth had no token → req.user was null → is_own always
false → Report button shown on the user's own reviews.

authRequest() when a token exists.
